# Contributing to BrainKit Starter

BrainKit Starter is the free foundation of the BrainKit system. Contributions are welcome.

## What to contribute

- Vault folder structure improvements
- CLAUDE.md routing patterns that work well for different roles or workflows
- `setup.sh` improvements for edge cases on macOS
- Documentation fixes and clarifications

## What not to contribute

- Changes that break the basic folder structure (paid clients depend on it being stable)
- Platform support beyond macOS — Windows and Linux are out of scope for v1

## How to contribute

1. Fork this repo
2. Create a branch describing what you're changing (e.g. `improve-setup-error-handling`)
3. Open a PR with a clear description of what problem it solves

## Issue tracker

Use [GitHub Issues](https://github.com/sockshead/brainkit-starter/issues) to report bugs, suggest improvements, or ask questions.
